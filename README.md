# The Good Fight TTRPG

Browser-based game tracker for *The Good Fight* solo mini-TTRPG. Open `index.html` and play — no installs or dependencies required.

PDF game rules at: [The Good Fight Mini-TTRPG (Solo).pdf](./The%20Good%20Fight%20Mini-TTRPG%20(Solo).pdf)

## Development

Test-driven development approach prior to implementation. Tests run in-browser (`tests/tests.html`) or via terminal (`npm test`; requires Node + happy-dom).

## Simulation

Simulation layer for automated playthroughs — single games or thousands at a time. All data collected and logged for balancing analysis, statistics, strategy metrics, etc.
